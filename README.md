# Compliment Generator

A simple, fun, and heartwarming web app that generates random compliments to brighten your day.

##  Features

- Generates a random compliment when you click the "Generate" button
- Compliments are deep, positive, and uplifting
- Prevents the same compliment from appearing twice in a row
- Smooth floating animation for the title
- Responsive and simple design

##  Project Structure
TheComplimentMachine
  -index.html # Main HTML file
  -style.css # CSS styles
  -script.js # JavaScript logic
  -README.md # This file

##  How to Use

1. Open `index.html` in your web browser
2. Click the **Generate** button
3. Enjoy a new compliment every time you click!
or go to https://aloysboscariol.github.io/TheComplimentMachine/

## Technologies Used

- HTML
- CSS (with simple animations)
- JavaScript (for dynamic content)
